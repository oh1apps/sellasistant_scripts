# Mapowanie produktów z Sellasist na ID w Shoperze

Skrypt pozwoli mapować produkty z Shopera na produkty w Sellasist. 

## Instalacja i obsługa:

1. Wrzuć plik na serwer
2. Dostosuj zmienne
3. Przeglądarkę skieruj na adres skryptu
